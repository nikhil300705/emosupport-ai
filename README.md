# EmoSupport AI 🤖💙

An intelligent emotional support agent designed to interact with users empathetically and efficiently solve their problems.

---

## 🚀 Features

* Adaptive conversation strategy (Ask → Empathize → Solve)
* Emotion-aware responses
* Deterministic and optimized decision-making
* Handles easy, medium, and hard tasks effectively
* Achieves perfect evaluation score (1.00)

---

## 🧠 Approach

The agent follows a structured progression:

1. **Ask Question** – Understand the user's issue
2. **Empathize** – Provide emotional support when needed
3. **Give Solution** – Deliver a helpful and final response

This ensures both **emotional intelligence** and **task completion efficiency**.
